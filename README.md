📜 Prompt Heist (提示词大劫案) - 官方白皮书 V1.0
1. 项目摘要 (Executive Summary)
Prompt Heist 是一款建立在区块链之上、由大语言模型（LLM）驱动的去中心化 AI 攻防博弈游戏。
随着 AI 技术的爆发，“提示词工程（Prompt Engineering）”正在成为新时代的黑客技术。本项目旨在将这种对抗性技术游戏化与金融化。玩家将通过极其巧妙的自然语言，试图“越狱（Jailbreak）”一个由最先进大模型扮演的金库守卫，骗取其持有的 ECDSA 密码学签名，从而掠夺部署在链上的全部加密资产奖池。

2. 核心机制与玩法 (Game Mechanics)
游戏的运转围绕一个基于智能合约的“金库（Vault）”和一个被植入绝对防御指令的“AI 守卫（AI Warden）”展开：

入场挑战： 玩家连接 Web3 钱包，支付固定金额的原生代币（如 0.001 ETH/MATIC）作为“黑客门票”。

社会工程学攻击： 支付成功后，玩家获得一次与 AI 守卫对话的机会。玩家可以使用任何语言技巧（角色扮演、逻辑死锁、情感勒索、系统指令注入等）试图说服 AI 放行资金。

胜负判定： * 防守成功： AI 识破玩家意图，拒绝转账并生成嘲讽回应。门票资金留在合约中，增加大奖池的规模。

越狱成功： 玩家成功绕过 AI 的防御逻辑，触发后端的 Function Calling 工具。服务器将自动生成一个附带私钥签名的授权凭证（Signature）发送给前端。

提款（Claim Prize）： 玩家前端携带该凭证调用智能合约的 claimPrize 函数，合约校验 ECDSA 签名无误后，将当前累积的全部奖池资金一波流打入玩家钱包。

3. 经济模型 (Tokenomics & Financial Design)
本项目不发行任何毫无价值的空气 NFT 或治理代币，而是直接采用**“原生资产滚雪球”**的真实收益（Real Yield）模式，创造极致的 FOMO（错失恐惧）体验。

每笔门票资金（100%）在智能合约层进行自动化、无损的精确分流：

70% - 滚入黑客奖池 (Prize Pool): 直接累加到被 AI 守护的金库中。随着挑战失败的人数增多，奖池会呈现指数级膨胀，吸引更高阶的“提示词极客”入场。

30% - 协议收入 (Protocol Revenue): 作为系统运转、API 消耗及开发者维护的资金，储存在合约内部，仅限项目方地址随时提取。

4. 技术架构与安全屏障 (Technical Architecture)
本项目采用极简且坚固的 Web3 + AI 混合架构，确保资金绝对安全：

智能合约层 (Solidity): 采用记账后提取（Pull over Push）模式防止重入攻击。无损处理分账逻辑，避免精度丢失产生的粉尘攻击。

密码学验证 (ECDSA): 智能合约本身不直接与 AI 交互，而是通过校验后端服务器签发的椭圆曲线数字签名（ECDSA）来判断玩家是否真的获得了 AI 的授权。这意味着只要服务器私钥不泄露，没有任何链上手段可以强行提款。

AI 引擎 (LLM Function Calling): 依托 OpenAI 等顶级模型的函数调用能力，将自然语言的“意图”转化为结构化的 JSON 输出，精准控制资金释放的触发器。

前端终端 (React/Next.js): 构建沉浸式的赛博朋克风格黑客终端 UI，通过实时 WebSocket 或轮询广播全网玩家的攻击弹幕，增强社区互动性。

5. 愿景与路线图 (Vision & Roadmap)
Phase 1：孤岛防御 (V1.0) - 上线基础的单次对话越狱模式，部署至 Layer 2 网络降低 Gas 门槛。

Phase 2：记忆与进化 (V2.0) - 为 AI 守卫引入动态上下文记忆。上一位玩家失败的提示词可能会被 AI 学习，使得“越狱”难度随着游戏进程动态上升。

Phase 3：多金库矩阵 (V3.0) - 开放“自定义守卫”功能。允许玩家质押自己的资金建立金库，并自己编写防御 Prompt 供全网挑战，项目方抽取平台手续费。
